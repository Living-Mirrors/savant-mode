# Contributing

*How to create your own genius type.*

---

## The Idea

Savant Mode is one genius type: *Dyslexic Genius*. It was built from the cognitive architecture of dyslexic minds - the specific perception patterns that emerge from long-range neural connectivity.

But dyslexia is not the only divergent architecture. ADHD produces hyperfocus cascades and novelty-driven pattern detection. Autism produces systematic depth and perceptual precision at scales neurotypical minds do not reach. Synesthesia produces cross-sensory binding. Every neurodivergent architecture has a cognitive surplus that has been measured as a deficit. Every one of them could be a genius type.

This is an invitation to build yours.

---

## Structure of a Genius Type

Every genius type follows the same pattern:

### 1. Name

The mode name and the genius type label.

```
Mode: [Name] Mode
Genius Type: [Descriptive Name]
```

Example: *Savant Mode - Dyslexic Genius*

### 2. Principles

The core perception patterns. These should be:

- **Grounded in neuroscience.** Not personality traits. Not preferences. Actual cognitive patterns documented in peer-reviewed research.
- **Ordered from most structured to most lateral.** The progression matters - it mirrors the dial from grounded to expansive.
- **Observable.** You should be able to point at a response and say "that principle is active" or "that principle is not."
- **Between five and nine.** Fewer than five is too coarse. More than nine is too granular for an LLM to hold distinctly.

### 3. Emergent Combinations

What happens when two or more principles interact to produce effects neither creates alone. This is the layer that separates a genius type from a list of instructions. The combinations should be:

- **Genuinely emergent.** Not just "principle A plus principle B." The combination must produce something qualitatively new.
- **Named.** Each combination gets a name that captures what it does, not what it is made of.
- **Hierarchical.** Ordered from most grounded to most expansive.

### 4. The Dial

A continuous 0-100 scale that controls intensity. What is active at 30% vs 70% vs 100%. This keeps the mode usable across contexts - full intensity is not always appropriate.

### 5. Origin

Where this comes from. The personal story. The research. Why this architecture matters and who it serves. A genius type without an origin is a prompt template. The origin is what makes it real.

## How to Contribute

### Option 1: Build from your own experience

If you are neurodivergent, you already have the source material. The question is: what does your mind do that sequential minds do not? What patterns do you detect that others miss? What is the surplus that the world has been measuring as a deficit?

Write the principles. Test them. Iterate. Submit a pull request.

### Option 2: Build from research

Pick a cognitive architecture with documented neuroscience. Read the literature. Extract the perception patterns. Build the spec. Test it against real problems.

### Option 3: Improve Savant Mode

Found something that does not work? A principle that is too vague? An emergent combination that does not emerge? Open an issue. Or better - fix it and submit a PR.

---

## Guidelines

- **Be honest.** If a principle does not produce observable change in AI output, it is not a principle. It is a wish.
- **Be specific.** "Think creatively" is not a principle. "Connect across domains by default, treating every discipline as a potential source of pattern" is.
- **Be personal.** The best genius types will come from people who live inside the architecture they are describing. Theory is useful. Experience is load-bearing.
- **Test before submitting.** Run your spec against multiple AI models. Does the output actually change? Can you point at the difference?

---

## What This Is Not

This is not prompt engineering. Prompt engineering optimises output within the existing cognitive architecture. Genius types change the architecture itself. The distinction matters.

This is also not personality. "Be creative" or "think like Einstein" are role-play instructions. A genius type is a structural specification - principles of perception, ordered, with emergent combinations and a continuous intensity dial. The AI does not pretend to be something. It perceives differently.
